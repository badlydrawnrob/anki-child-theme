We call a a Type Variable. Think of type variables as a placeholder for a specific Type.

You’ll often see a as a type variable, which stands for any Type. The type a can vary depending on how the function is used, but once we decide what a is, that’s what it is everywhere.

Type variables represent concrete types that have not been specified yet. For example number is a constrained type variable. It can only be an Int or a Float.
